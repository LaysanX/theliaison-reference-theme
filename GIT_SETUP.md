# Git Setup

This reference theme is ready to publish as a standalone Git repository.

Recommended public repository:

```bash
https://github.com/laysanx/theliaison-reference-theme
```

Initial publish commands:

```bash
git init
git add .
git commit -m "Initial TheLiaison reference theme"
git branch -M main
git remote add origin https://github.com/laysanx/theliaison-reference-theme.git
git push -u origin main
```

Designer clone command:

```bash
git clone https://github.com/laysanx/theliaison-reference-theme.git
```

Keep this repo frontend-only. Do not add .NET, server-side scripts, secrets, exported production uploads, or client data.

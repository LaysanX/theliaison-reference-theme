# Designer Release Notes - v2.0.0

This release upgrades the TheLiaison reference theme from a starter package into the master designer reference for the current LaysanX uploaded-theme runtime.

## What Is New

- Added full ecommerce/customer template coverage for cart, checkout, order success, order lookup, returns, account, addresses, orders, wishlist, profile, and change-password.
- Added structured reference docs for feature coverage, template mapping, data objects, platform-managed surfaces, and visual design patterns.
- Expanded `theme.json` to reflect the real runtime-supported template surface and current theme setting inputs.

## Package Cleanup

- Removed repository-only handoff clutter such as embedded git metadata and local junk files.
- Rebuilt the downloadable ZIP from the cleaned source package.

## Why Designers Should Use This Version

- It matches the current uploaded-theme runtime more accurately than the earlier starter package.
- It clarifies which public pages are themeable today and which customer auth flows are still platform-managed.
- It gives a stronger design-system style reference for headers, listings, collections, product detail, checkout, and account experiences.
